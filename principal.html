¿Qué es GIT?

Git es un sistema de control de versiones, el cuál es un proyecto de código abierto y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005.
Está pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

¿Qué es un sistema de control de versiones?

El SCV o VCS (por sus siglas en inglés), son sistemas de control que ayudan a realizar un seguimiento y registro de los cambios realizados a uno a varios archivos a lo largo del tiempo. Además registran datos extras como quién lo editó, la fecha de modificación, etc.




Características de Git

  1. Guarda cambios de forma incremental.
  2. Al contar con un historial, permite seleccionar una versión anterior y agregar funcionalidades.
  3. Lleva un registro de los cambios por persona para cada archivo.
  4. Aunque fue diseñado para operar en un entorno Linux, actualmente, es multiplataforma.
  5. Funciona y se instala en la máquina de manera local y se recomienda su uso desde una terminal o línea de comandos.
